# **App Name**: Whisper Rooms

## Core Features:

- User Authentication: Secure user login/registration using Firebase authentication.
- OTP Verification: Implement OTP verification for enhanced security during registration.
- Room Listing: Display a list of available chat rooms.
- Room Creation: Allow users to create new chat rooms.
- Voice Chat: Enable real-time voice chat functionality within chat rooms.
- User Profiles: Enable users to create and manage their profiles.

## Style Guidelines:

- Primary color: Calm blue (#77B5FE), suggesting communication and tranquility.
- Background color: Very light blue (#F0F8FF) for a clean and airy feel.
- Accent color: Soft lavender (#E6E6FA) for subtle highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif, giving a modern, machined, objective, neutral look.
- Use minimalist line icons to represent chat functions and user actions.
- Clean, card-based layout for rooms and profiles, with consistent spacing and alignment.
- Subtle transitions when navigating between rooms and profiles.
